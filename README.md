# mcp-filesystem# INPI Trademark Analyzer

# 

# Sistema automatizado de triagem preliminar de colidência marcária.

# 

# Funcionalidades:

# \- Busca automática no INPI (PEPI)

# \- Análise de similaridade textual e fonética

# \- Classificação de risco (LOW / MEDIUM / HIGH)

# \- Geração de relatório estruturado

# \- Minuta automática de parecer jurídico preliminar

# 

# Objetivo:

# Auxiliar escritórios de Propriedade Intelectual na análise inicial de registrabilidade de marcas,

# reduzindo tempo operacional e padronizando critérios de avaliação.

# 

# Tecnologias:

# Python + Playwright + Parsing dinâmico do portal INPI
